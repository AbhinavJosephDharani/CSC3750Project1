Courses & Projects

CSC 3110 - Algorithm Design and Analysis

This course focused on evaluating algorithm efficiency using asymptotic analysis. Topics included:

  • Divide-and-Conquer
  • Greedy Techniques
  • Dynamic Programming
  • Time-Space Tradeoffs
We also covered NP-completeness and approaches for intractable problems.

CSC 4500 - Introduction to Theoretical Computer Science

This class explored the theory of computation and formal languages. Key concepts included:

  • Finite Automata, CFGs, and PDAs
  • Turing Machines and Undecidability
  • Pumping Lemmas and Language Hierarchies
The course built foundational knowledge for understanding the limits of computation.
